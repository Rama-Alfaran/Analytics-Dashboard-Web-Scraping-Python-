Analytics Dashboard & Web Scraping (Python)
Overview

This project presents a data analytics dashboard built on vendor data collected through web scraping from company websites. It demonstrates the use of data processing techniques and visual analytics to extract meaningful insights and support data-driven decision making.

Features
Web scraping of vendor data from multiple company websites
Data cleaning and preprocessing for analysis
Application of data analytics techniques
Interactive and visual dashboards (charts/graphs)
Insight extraction for better understanding of trends and patterns
Technologies Used
Python
Web Scraping (e.g., BeautifulSoup / Selenium)
Data Analysis (Pandas, NumPy)
Data Visualization (Matplotlib / Seaborn / Plotly)
